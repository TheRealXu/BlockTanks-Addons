1. Variables and Data Types

What are Variables?

Variables store data that your program can use and manipulate. They are declared using var, let, or const. Hereâ€™s a breakdown:

  • var: Function-scoped. Avoid using it in modern JavaScript.
  • let: Block-scoped, introduced in ES6, and allows reassignment.
  • const: Block-scoped, introduced in ES6, and does not allow reassignment.
// Example 1: Variable declarations
let name = 'Alice';  // A string
const age = 25;     // A number
var isStudent = true;  // A boolean

// Example 2: Reassigning variables
let mood = 'happy';
mood = 'excited';  // Allowed with let

// Example 3: Constants
const pi = 3.14159;
// pi = 3.14; // Error: Constants cannot be reassigned

Data Types in JavaScript

JavaScript has several data types divided into two categories:

  • Primitive Types: string, number, boolean, undefined, null, symbol, and bigint.
  • Non-Primitive Types: object and array.
// Example 1: Primitive types
let message = 'Hello';  // String
let count = 10;         // Number
let isAvailable = true; // Boolean

// Example 2: Non-primitive types
let person = { name: 'Alice', age: 25 };  // Object
let colors = ['red', 'blue', 'green'];  // Array

Dynamic Typing

JavaScript is dynamically typed, meaning variables can change their data type at runtime.

// Example: Dynamic typing
let value = 'Hello'; // Initially a string
value = 42;         // Now a number
value = true;      // Now a boolean

2. Functions

What Are Functions?

Functions are reusable blocks of code that perform specific tasks. They are defined using the function keyword or as arrow functions in ES6.

// Example 1: Function declaration
function sayHello() {
    console.log('Hello!');
}

// Example 2: Function with parameters
function add(a, b) {
    return a + b;
}

// Example 3: Arrow function
const multiply = (a, b) => a * b;

// Calling the functions
console.log(add(3, 5)); // Output: 8
console.log(multiply(4, 6)); // Output: 24

Anonymous Functions

Anonymous functions have no name and are often used in callbacks or as arguments to higher-order functions.

// Example: Anonymous function as a callback
setTimeout(function() {
    console.log('This runs after 2 seconds.');
}, 2000);

Function Scope

Variables declared inside a function are only accessible within that function. This is known as function scope. 

// Example: Function scope
function testScope() {
    let message = 'Inside function';
    console.log(message);  // Accessible here
}
// console.log(message); // Error: message is not defined

3. Control Flow

Conditional Statements

Control the flow of your program using conditional statements like if, else, and switch.

// Example: if-else
let age = 20;

if (age >= 18) {
    console.log('You are an adult.');
} else {
    console.log('You are a minor.');
}

// Example: switch
let fruit = 'apple';

switch (fruit) {
    case 'apple':
        console.log('You chose an apple.');
        break;
    case 'banana':
        console.log('You chose a banana.');
        break;
    default:
        console.log('Unknown fruit.');
}

Loops

Loops repeat a block of code until a specified condition is met. Common loops include for, while, and do-while. 

// Example: for loop
for (let i = 0; i < 5; i++) {
    console.log(i);
}

// Example: while loop
let count = 0;
while (count < 5) {
    console.log(count);
    count++;
}

// Example: do-while loop
let num = 0;
do {
    console.log(num);
    num++;
} while (num < 5);
